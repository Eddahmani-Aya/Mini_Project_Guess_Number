# 🎮 Guess Number Game (Java)

## Description
This is a simple Java game where the player tries to guess a randomly generated number.

## How it works
- The program generates a random number
- The player enters guesses
- The program gives hints (higher / lower)
- The game continues until the correct number is guessed

## Objectives
- Practice PHP basics
- Use loops and conditions
- Work with user input
- Understand random number generation
